What is a Feed?

A blog is the most common type of Feed. There are other kinds of Feeds, such as perhaps weather reports, news bulletins, price notices, etc. Essentially a Feed is a bunch of Articles on a certain topic. In a way, Feeds are the most important element of BlogBridge. Here are the key ideas about a Feed. It has name and a one or more articles. Many Feeds have a corresponding web site where you can read the blog or feed with a conventional web browsers. It also has URL, which refers to the feed in it's raw form, either RSS or ATOM. While this is a crucial piece of information, you generally don't have to think about it.

It is important to understand that all the information in a Feed in provided by the Feed author, either a blogger or a news site. BlogBridge can only read feeds. Nothing you do in BlogBridge will affect what other people see. BlogBridge is there to help you quickly discover, read and keep track of Feeds.

Feed Properties

The Feed Properties Dialog Box displays the following information about the Feed. There are the following "Basic" Properties:

  • Name: The Name of the Feed. You can edit this for your own convenience, but of course change won't affect anyone else.
  • Description: A paragraph of further information, supplied by the Feed author. This too can be locally edited
  • Site URL: The URL for the web site corresponding to the Feed
  • XML URL: The URL for the RSS or ATOM flow of articles.
  • Language: The Language (e.g. French or Papiamentu) in which the Feed is written.

Then there are the following "Activity" Properties:

  • Articles: The number of Articles which BlogBridge currently has from this Feed
  • Retrievals: The number of times that BlogBridge has retrieved new articles
  • Last Update: The date of the last time the Feed was updated by the author
  • Format: The technical format of the data in the Feed

Finally we have the following "BlogStarz" Properties:

  • Technorati Inlinks: The number of other Feeds which refer or link to this one. This is an excellent way to tell how "authoritative" or "important" this feed is.
  • Activity: A number between 0.0 and 1.0 indicating how frequently this Feed is updated
  • Inlinks: A number between 0.0 and 1.0, based on the Technorati Inlinks
  • Keywords: A number between 0.0 and 1.0 indicating how many Keyword hits this Feed has.
  • BlogStarz Rating: A rating, between 1 and 5 Stars computed by BlogBridge, which will tell you how much we think you will like this Feed
  • Final Rating: Your own Rating of this Feed, if you gave it one, or the BlogStarz rating if you haven't yet rated this one yourself. Note that there's a button next to the Final Rating, allowing you to erase your own rating and use the BlogStarz rating.
