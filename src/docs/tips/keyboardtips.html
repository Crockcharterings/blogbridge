Keyboard Shortcuts

Globally Available
CTRL-N Subscribe to new Feed.
CTRL-1 - CTRL-9 Fast Guide switching.
CTRL-SHIFT-ALT-M Update all Guides.
CTRL-Up / CTRL-Down Move selected Guide up / down.
CTRL-F1 Service dialog.
CTRL-F5 Load guides from service.
CTRL-F9 Store guides on service for future loading.
SPACE / SHIFT-SPACE Mark article as read and select next / previous unread article.
CTRL-SPACE Mark currently selected feed as read and move to the next unread article.
K Select next article with keywords found.
CTRL-SHIFT-P Show Preferences dialog.
CTRL-SHIFT-F10 Show Networking Manager.
CTRL-"-" Make articles text font smaller.
CTRL-"+" Make articles text font bigger.
1 - 4 Article filters: All, Unread, With Keywords, Pinned
5 - 7 Article view mode: Minimal, Brief, Full
Guides List
Insert Add new Guide.
Delete Delete selected guide.
CTRL-SHIFT-M Update selected guide.
Feeds List
CTRL-M Update selected feed.
ALT-Up / ALT-Down Move feed up and down in the list. Note that the behaviour can be affected by feeds sorting seeting.
Insert Subscribe to new Feed.
Delete Delete selected feed.
CTRL-SHIFT-Q Mark selected feed as read.
CTRL-P Show feed properties dialog.
T Show feed tags dialog.
CTRL-SHIFT-B Browse the selected feed.
Articles List
Q Toggle article read / unread.
CTRL-Q Mark selected article as read.
CTRL-SHIFT-Q Mark whole feed as read.
Up / Down Scroll the list several lines upper / lower.
Page Up / Page Down Scroll the list one page upper / lower.
CTRL-Up / CTRL-Down Select upper / lower article.
T Show article tags dialog.
P Pin / unpit the selected article.
B Create a blog entry for the selected article.
CTRL-B Show an article in a browser.
1 - 4 Article filters: All, Unread, With Keywords, Pinned
5 - 7 Article view mode: Minimal, Brief, Full
Left / Right Cycle through view modes for individual article
CTRL-Left / CTRL-Right Cycle through view modes for all articles in a feed
